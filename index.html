Proyecto de Home banking - Entrega final


Se implementan las funciones de estas páginas

para cumplir los objetivos de la entrega.


Se usa la librería luxon.js para colocar la fecha al pié de página.

Se usa la librería sweetalert2 para mostrar los mensajes de error en el formulario de identificacion.

Se leen archivos .json , tanto para usuarios como para monedas.


Las funciones en Java Script:

En todas las paginas se verifica que exista un usuario identificado.

Una variable de sessionStorage se carga cuando pasa por index.html

Si no se verifica, lo redireccionamos a inicio.html para que se identifique


En index.html pedirá un nombre de usuario y password válidos

Los usuarios, password y Nombre de usuario válidos fueron cargados

en un archivo .json para simular la lectura de una DB

Los usuarios posibles son:

   "u1", "p1"

   "usuario1", "password1"

   "usuario2", "password2"

   "usuario3", "password3"


Con Javascript se verifica que sean correctos

Si son correctos nos vamos a ver los servicios del Home Banking

y si son incorrectos se muestra un mensaje de Error


En index.html, al ingresar por primera vez, se carga la tabla de usuarios

en una sessionStorage para ser utilizado durante toda la session


Las divisas posibles, como así los valores de compra y ventas se

cargaron en un archivo .json para simular la lectura de una DB

Con los elementos leídos en el .json armamos la página de "Cambio de divisas
